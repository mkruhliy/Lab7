Система реєстації та авторизації для роботи з програмою Notebook

# Список користувачів - accounts.txt; перевір чи файл знаходиться в робочій директорії


1)Реєстрація
(def register)

# При реєстрації імені, що починається з 'admin' користувач має можливість банити/розбанити інших користувачів

Вийнятки при реєстрацї:
- Не вказано ім'я(NoUsername)
- Не вказано пароль(NoPassword)
- Ім'я вже зарезервовано(UserNameAlreadyExists)
- Пароль надто короткий(PasswordTooShort)

# Після реєстрації автоматичний перехід до форми авторизації


2)Авторизація
(def auth)

Вийнятки при авторизації:
- Не вказано ім'я(NoUsername)
- Не вказано пароль(NoPassword)
- Неіснуюче ім'я(InvalidName)
- Невірний пароль(InvalidPassword)
- Користувач забанений(Banned)


+ Вийнятки при бані/розбані користувачів
- Невірно вказана дія забанити/розбанити(InvalidNumber)
- Вказано неіснуючий аккаунт для бану/розбану або користувача з іменем 'admin'(InvalidName)
- Спроба бану забаненого користувача(Banned)
- Спроба розбану не забаненого користувача(NotBanned)

notebook.py - алгоритм записника
auth_account.py - алгоритм реєстрації та авторизації
main.py - інтерфейс користуача
